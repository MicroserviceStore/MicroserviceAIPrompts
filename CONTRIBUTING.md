# Contributing to Embedded Microservice Prompts

First off, thank you for taking the time to contribute! Community inputs are what keep this repository accurate, efficient, and valuable for embedded engineers worldwide.

By contributing to this project, you agree that your contributions will be licensed under the project's [Apache 2.0 License](LICENSE).



## Pull Request Guidelines

To maintain repository quality, please ensure your PR follows these steps:

1. Open an **Issue** describing the problem and the AI output, or a new prompt creation.
2. **Fork the Repository:** Create a feature branch for your changes.
4. **Submit the PR:** Provide a clear description of what the prompt accomplishes or fixes. 

Thank you for helping build a smarter ecosystem for embedded developers!
