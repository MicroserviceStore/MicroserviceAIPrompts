# Embedded Microservice Prompts

Welcome to the **Embedded Microservice Prompts** repository, by Microservice Store (www.microservicestore.com)! This is a community-driven collection of optimized AI prompts designed to generate reliable, modular, and lightweight microservices for embedded systems, from core template to vertical specific (connectivity, cryptography, storage etc).

Our goal is to help embedded developers accelerate firmware architecture design using generative AI while maintaining strict standards for memory efficiency, safety, and concurrency.

---

## 🚀 How to Use These Prompts

1. Go to Microservice Store Product Dashboards, and select a product to create a Microservice using AI.
2. Click "Generate By AI"
3. Microservice Store uses the `/prompts/microservice_gen.md` by default, to pass the essential details about an embedded Microservice Generation.
4. The developers then pass their custom requests in the same window to generate a Microservice by AI.
5. This repository also collects vertical-specific prompts under `/prompts/` organised under directories, such as `crypto`, `connectivity`, where the developer can select a specific prompt for more specific code generations.
